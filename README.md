# SpellTome
A hand-tailored spell tome for GrimfallWoW, attempting to list all available spells and talents to the player.

**Disclaimer:** Everything you see in this AddOn was added by hand. If you see any discrepancies, or spells that are missing/wrong, please leave an Issue on https://github.com/harukaze3/SpellTome/issues or in the GrimfallWoW Addons Discord server (https://discord.gg/Qx4tjErvbe)

## Functions
- What SpellTome does:
  - SpellTome attempts to list every single ability and talent you can learn from 1 to 60 in a neatly ordered fashion.
  - Acts as a pseudo-database. (Every entry is provided via an ID, so if the tooltips ever change but the IDs don't, it should be automatically reflected without having to update the addon.)

- What SpellTome doesn't do:
  - SpellTome is not your spellbook. You can't cast spells from it, you can't drag spells from it onto your action bars. It is purely a browser.
  - Doesn't spoof any of the gamefiles, so no risk of getting the addon banned or worse, getting yourself banned for using it.
